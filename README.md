# LAP 4 CODING CHALLENGE

## Installation
* Clone repo
* cd into the folder - `cd lap4_url_shortner`
* `pipenv shell`
* `pipenv install`
* locally - `pipenv run dev` running on http://127.0.0.1:5000 

## Deployment
* ShortyUrly deployed to Heroku - https://shortyurly.herokuapp.com/
